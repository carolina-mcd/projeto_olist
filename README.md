
# 📊 Análise de Vendas — Olist E-commerce

## 📌 Sobre o Projeto
Análise exploratória dos dados do e-commerce brasileiro Olist, 
com o objetivo de identificar padrões de vendas, comportamento 
de entrega e perfil dos consumidores.

## 🗂️ Dados Utilizados
- Dataset público disponível no [Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- 99.441 pedidos entre 2016 e 2018
- 9 arquivos CSV com informações de pedidos, clientes, produtos e pagamentos

## 🔍 Principais Insights

### 📦 Pedidos e Receita
- Volume de pedidos cresceu consistentemente ao longo de 2017 e 2018
- Receita acompanhou o crescimento no volume de pedidos

### 🏆 Categorias
- As categorias com maior receita são cama_mesa_banho, beleza_saude e informatica

### 🚚 Entregas
- Tempo médio de entrega real: **12.1 dias**
- Tempo médio estimado: **23.4 dias**
- **91% dos pedidos entregues antes do prazo**

### 🗺️ Regiões
- São Paulo lidera em volume de pedidos e receita
- Sudeste concentra a maior parte das vendas

### 💳 Pagamentos
- Cartão de crédito é a forma de pagamento dominante
